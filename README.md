# backgammon
Jeu de backgammon

## Liste des entités

### PLATEAU - BOARD
- flèches 6 x 2 x 2

### JOUEUR - PLAYER
- sens
- 15 pions
- couleur
- score

### PIONS - Checker
- position (numéro flèche ?)
- etat : barre / en cours / dernière case /
- couleur / joueur

### FLECHE - Points
- couleur
- état : fermée / pas fermée
